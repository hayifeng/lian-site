# 前端任务列表 (F1-F23)

## 技术栈要求
- React 18+ with TypeScript
- Vite 作为构建工具
- Tailwind CSS 3+ 用于样式
- Framer Motion 或 React Spring 用于2D动画
- Three.js + @react-three/fiber + @react-three/drei 用于3D效果
- Axios 或 Fetch API 用于HTTP请求

## 任务列表

### F1: 初始化前端项目
**输入**: 无  
**输出**: `app/frontend/` 目录包含基础项目结构（package.json、tsconfig.json、vite.config.ts、index.html等）
**依赖关系**: 无
**说明**: 使用 Vite React+TypeScript 模板创建项目，配置基本设置

### F2: 配置开发环境与工具链
**输入**: F1 的输出
**输出**: 配置好的 ESLint、Prettier、TypeScript 严格模式、路径别名
**依赖关系**: F1
**说明**: 添加代码质量工具，配置开发环境

### F3: 安装核心UI与动画库
**输入**: F1 的输出
**输出**: package.json 中包含所有必要依赖
**依赖关系**: F1
**说明**: 安装 Tailwind CSS、Framer Motion、Three.js 相关库、React Router

### F4: 创建基础布局与路由
**输入**: F2, F3 的输出
**输出**: 基础布局组件、四个页面的路由配置（React Router）
**依赖关系**: F2, F3
**说明**: 创建 App.tsx、Layout 组件，配置四个页面的路由路径

### F5: 实现环形展示页面（布局与静态图片）
**输入**: F4 的输出
**输出**: 环形布局的静态版本组件，图片均匀分布在圆形轨道上
**依赖关系**: F4
**说明**: 使用 CSS/Canvas 实现圆形轨道布局，显示静态图片占位符

### F6: 实现环形展示页面（旋转动画与滚动交互）
**输入**: F5 的输出
**输出**: 支持鼠标滚轮控制轨道旋转的动画逻辑
**依赖关系**: F5
**说明**: 实现平滑的旋转动画，鼠标滚轮向上/向下对应顺时针/逆时针旋转

### F7: 实现环形展示页面（悬停与点击交互）
**输入**: F6 的输出
**输出**: 图片悬停高亮放大效果，点击事件绑定
**依赖关系**: F6
**说明**: 悬停时图片放大、轻微脱离轨道，点击可触发详情页跳转（暂用控制台日志）

### F8: 实现3D轮播页面（布局与静态卡片）
**输入**: F4 的输出
**输出**: 3D透视布局的静态卡片列表组件
**依赖关系**: F4
**说明**: 创建3D轮播的基础HTML结构，使用CSS 3D变换或Three.js

### F9: 实现3D轮播页面（自动轮播与景深效果）
**输入**: F8 的输出
**输出**: 卡片自动循环移动，非选中卡片有景深效果（尺寸减小、透明度增加）
**依赖关系**: F8
**说明**: 实现自动轮播动画，两侧卡片向后方退去，模拟空间层次

### F10: 实现3D轮播页面（滚动控制与悬停选中）
**输入**: F9 的输出
**输出**: 鼠标滚轮控制左右移动，悬停卡片上浮高亮
**依赖关系**: F9
**说明**: 滚轮控制画廊移动，悬停时卡片Y轴上浮、亮度提升

### F11: 实现泡泡头像页面（布局与静态头像）
**输入**: F4 的输出
**输出**: 自由布局的静态头像列表组件
**依赖关系**: F4
**说明**: 创建不规则排列的圆形头像网格，使用CSS Grid或绝对定位

### F12: 实现泡泡头像页面（入场动画与浮动效果）
**输入**: F11 的输出
**输出**: 页面加载淡入动画，头像上下浮动动画
**依赖关系**: F11
**说明**: 使用Framer Motion实现渐显效果和轻微浮动动画

### F13: 实现泡泡头像页面（悬停与焦点效果）
**输入**: F12 的输出
**输出**: 悬停头像放大清晰化，其他头像变暗模糊
**依赖关系**: F12
**说明**: 悬停时头像缩放1.2-1.5倍，降低模糊度，其他头像降低不透明度

### F14: 实现泡泡头像页面（视差滚动与动态加载）
**输入**: F13 的输出
**输出**: 滚动视差效果，滚动到底部动态加载更多头像
**依赖关系**: F13
**说明**: 头像层与背景层不同滚动速度，实现无限滚动加载

### F15: 实现照片管理页面（布局与基础UI）
**输入**: F4 的输出
**输出**: 管理页面布局，包含上传区域、图片列表区域、操作按钮
**依赖关系**: F4
**说明**: 创建基础管理界面，使用Tailwind设计组件

### F16: 实现照片管理页面（上传组件）
**输入**: F15 的输出
**输出**: 支持拖拽和文件选择的图片上传组件
**依赖关系**: F15
**说明**: 实现文件选择、拖拽区域、上传进度显示

### F17: 实现照片管理页面（图片列表与删除）
**输入**: F16 的输出
**输出**: 显示已上传图片的列表，每个图片有删除按钮
**依赖关系**: F16
**说明**: 图片网格布局，删除按钮点击触发删除操作

### F18: 实现认证UI（登录/注册表单）
**输入**: F4 的输出
**输出**: 登录和注册表单组件，包含输入验证
**依赖关系**: F4
**说明**: 创建美观的表单，处理输入验证，显示错误信息

### F19: 集成API客户端（axios/fetch封装）
**输入**: F2 的输出
**输出**: 统一的API请求工具，包含错误处理和拦截器
**依赖关系**: F2
**说明**: 创建 api.ts 文件，封装所有HTTP请求，处理token自动附加

### F20: 连接前端与后端API（照片管理）
**输入**: F17, F19 的输出，以及后端B4,B5,B6任务完成
**输出**: 照片上传、删除、列表获取功能与UI完全集成
**依赖关系**: F17, F19, B4, B5, B6
**说明**: 将上传组件连接到上传API，将删除按钮连接到删除API，加载图片列表

### F21: 连接前端与后端API（认证）
**输入**: F18, F19 的输出，以及后端B2,B3任务完成
**输出**: 登录/注册功能与后端集成，token存储，路由保护
**依赖关系**: F18, F19, B2, B3
**说明**: 表单提交调用认证API，存储JWT，实现受保护路由

### F22: 响应式设计与样式优化
**输入**: 所有前端页面组件
**输出**: 适配移动端和平板，整体样式抛光
**依赖关系**: 所有前端任务
**说明**: 使用Tailwind响应式类优化所有页面，确保跨设备兼容

### F23: 前端测试（组件与交互）
**输入**: 所有前端组件
**输出**: 关键组件的单元测试和集成测试
**依赖关系**: 所有前端任务
**说明**: 使用Vitest+React Testing Library测试核心组件和交互逻辑

## 执行建议
1. 按编号顺序执行任务
2. 每个任务完成后验证输出是否符合要求
3. 涉及动画的任务先实现基础功能，再优化性能
4. API集成任务需要等待对应后端任务完成
